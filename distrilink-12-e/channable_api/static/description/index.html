Download the project API csv file.

Upload and import the api file (Only csv file formate) and it will create the products.

It wil be added the channable product details.

To see price list go to configuration > channable > Product price.

Orders from Channable will create in odoo when sync the order after every 15 minutes if there is any new order in channable

Orders From Channable will automatically confirm the sale order in odoo.

Invoice will also generate, validate and will be send to the customer through mail automatically.

It will create the Delivery order and you can validate throught the validate button. Shipment URL which is added to the channable information will create from which you can redirect to the url. It must filled with the chanable information otherwise validated order will not pass to the channable.

If any product does not exists in odoo then it will create a error order and sale order line of that product will display in red coloured line. Confirm button will not display on error order. Error orders have to do invoice, sending mail process and sale order confirm manually.

To confirm sale order, click Accept button.

Set logo and it will also print into the invoice.

Logo has been printed to the invoice.

Cancel the sale order from odoo will also cancelled that perticular order from Channable and display in the Cancelled.
