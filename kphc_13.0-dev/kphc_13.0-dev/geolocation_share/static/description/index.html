Share geolocation on shipping page

Ease customers pain caused by filling in all those fields at checkout

An efficient application which helps customers to pinpoint the exact location on the map so the delivery of the goods runs smoothly avoiding useless phone calls asking for directions. The application allows Geocoding Customer Addresses in Odoo via Google Maps. Geocoding is the process of finding associated geographic coordinates(latitude and longitude) from other geographic data.

Features:

  • Auto-detect location on the map as per customer address.
  • Auto-detect location on the map as per customer shipping/billing address during checkout.
  • The customer can select his/her address using google map during checkout.
  • A customer can select his/her address using google map to update existing address.
  • Automatically switches to the current position inside map when a customer updates his/her city/state/country.
  • Real-time address update when a customer changes his/her position by dragging or click on the map.
  • Get the position of the current location on a map when a customer adds a new address.
